# Local development

## For Server side App
```
cd server
```
### Project setup
```
npm install
```
#### Compiles and hot-reloads for development
```
npm start
```

## For client side App
```
cd client
```
### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
